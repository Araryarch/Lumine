LUMINE - QUICK START GUIDE
==========================

STEP 1: CLONE & BUILD
---------------------
git clone https://github.com/Araryarch/lumine.git
cd lumine
go mod download
go build -o lumine

STEP 2: RUN LUMINE
------------------
./lumine

Or using Make:
make dev

STEP 3: SETUP DATABASES (Optional)
-----------------------------------
In another terminal:
make db-setup

NAVIGATION
----------
Arrow Keys / j/k  - Move up/down
h/l              - Switch panels
Tab              - Next panel
Enter            - Select

ACTIONS
-------
Space   - Select service
s       - Start
x       - Stop
r       - Restart
v       - Change version
Ctrl+C  - Quit

EXAMPLE WORKFLOW
----------------
1. Run: make dev
2. Navigate to "Services"
3. Select MySQL (Space)
4. Press 's' to start
5. Done!

DATABASE ACCESS
---------------
MySQL:      mysql -h localhost -P 3306 -u root -proot
PostgreSQL: psql -h localhost -p 5432 -U postgres
MongoDB:    mongosh mongodb://root:root@localhost:27017
Redis:      redis-cli -h localhost -p 6379

ADMIN PANELS
------------
phpMyAdmin:      http://localhost:8080
Adminer:         http://localhost:8081
Mongo Express:   http://localhost:8082
Redis Commander: http://localhost:8083
pgAdmin:         http://localhost:8084
Caddy:           http://localhost:8085

MAKEFILE COMMANDS
-----------------
make dev       - Run in development mode
make build     - Build binary
make install   - Install to system
make db-setup  - Setup all databases
make db-stop   - Stop databases
make clean     - Clean build artifacts
make help      - Show all commands

TROUBLESHOOTING
---------------
Docker not running?
  - macOS: Open Docker Desktop
  - Linux: sudo systemctl start docker
  - Windows: Start Docker Desktop

Port conflict?
  - Lumine auto-finds alternative ports
  - Or check: lsof -i :3306

Build error?
  - make clean
  - make deps
  - make build
